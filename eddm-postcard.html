What is EDDM?

Every Door Direct Mail lets you send postcards to every address on a postal carrier route — no mailing list, no names, no addresses needed. You pick routes by ZIP code and USPS delivers to every door. Perfect for local service businesses.

How to Send via USPS EDDM

  1. Go to eddm.usps.com and click "Start a Campaign"
  2. Enter each ZIP code above and browse available carrier routes
  3. Filter routes by residential only (skip PO boxes and businesses if budget is tight)
  4. Select routes — each shows household count (typically 300-700 per route)
  5. Print postcards at a local print shop or online (VistaPrint, PrintingForLess, GotPrint)
  6. Bundle 100 postcards per bundle, face up, with a facing slip on top
  7. Drop bundles at your local post office with EDDM retail form (PS Form 3587)

Best Timing

  • March-April: Homeowners planning spring/summer projects — best ROI
  • August-September: "Get it done before winter" urgency
  • Avoid December-February (nobody painting exterior in New England winter)
  • Drop on Tuesday-Wednesday for midweek mailbox delivery

Recommended Strategy

  • Drop 1 (Spring): Wakefield + Melrose + Stoneham + Reading (~2,000-3,000 homes, ~$700-1,100)
  • Drop 2 (6 weeks later): Lynnfield + Saugus + Winchester + Woburn
  • Drop 3 (Late summer): Malden + North Reading + Medford + Peabody
  • Track which ZIPs generate calls — double down on those routes next season
